# About

Screen relative 3D model translation/rotation.

**Live Demo**
https://simple-voxel-demo.surge.sh/

**Preview**
![Banner](https://github.com/DevArt002/Simple-Voxel/blob/master/preview.gif)

# How to install

Run locally: `npm i` && `npm run dev`
Build, `npm run build`

# Technologies/Libraries

- [x] SPA using React/React Hooks
- [x] Three.js for 3D visualization
- [x] Vite
- [x] Typescript
- [x] ESLint
- [x] Prettier
- [x] Sass
- ...

# Implemented Features

- [x] Camera Controls (Left mouse: pan, Right mouse: rotate, Middle mouse: zoom)
- [x] Dropdown for mesh preset
- [x] Mesh creation by clicking on grid
- [x] Translation controller for translating mesh relative to screen
- [x] Rotation controller for rotating mesh relative to screen
- [x] Remove/Remove all
- [x] Tree view for exploring 3d scene hierarchy
